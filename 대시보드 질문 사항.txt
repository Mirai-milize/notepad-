1. 대시보드

디자인 관련
	해당 내용을 모두 포함한다면, 저희 측에서 디자인 제안을 드려도 괜찮을까요?
	제공해주신 와이어프레임 그대로 진행하면 빈 공간등이 많이 생길 것으로 사료됩니다.

[1] 자산 현황
	1) 전체-당월 필터 적용 범위
		해당 필터는 어느 항목에 반영되어야 하는 지 세부적인 내용이 궁금합니다.
		전체-당월 필터에 맞추어 하기한 내용이 전부 변경되어야하는지,
		특정 테이블만 병경된다면 어떤 부분이 변경되어야하는지 확정 부탁드립니다.
		
	2) 총 자산 가격 테이블 수식 관련
		1. 총 자산 가격 = 현재 보유한 차량 전체의 MSRP(신차가격) 합
		2. 매입가 = 현재 보유한 차량 전체의 매입가
		3. 할입금액 = 현재 보유한 차량 전체의 할인금액
		4. 할인율 = (할인금액 ÷ MSRP(신차가격)) × 100
		해당 사항으로 진행하면 되는지 확정 부탁드립니다.
		
[2] 비용 현황
	1. 대출 현황 테이블 관련
		1) 대출 스케줄표 
			스케줄표 도입 전까지는 신차가격과 금리 5.9%를 가정한 PMT 기반 추정값으로 표기하고, 스케줄표가 생성되는 시점부터는 스케줄표 기준 값으로 전환하는 방식으로 진행해야 할까요?
			또한 현재 기준으로는 대출이율이 변경되는 경우 (연도 변경등)를 대응하는 로직이 없어,
			대출 스케줄표 기능이 추가된 이후에 해당 기능이 완전히 구현될 것으로 생각됩니다.
			
	2. 취·등록세 테이블
		1) 해당 테이블에 대한 데이터 정의와 관리
			하기한 테이블의 세부 정의에 대한 확인 및 정의를 요청드립니다.
			a. 취·등록세 = 신차가격의 4%(취득세 2%, 등록세2%) 일괄 적용하여 총 합 표기
			b. 자동차 보험료 :
			c. 부대비용 :
			d. 수수료 :
		2) 증감 % 기준
			해당 내역의 증감은 어떤 데이터를 기준으로 설정해야하는 지 확정부탁드립니다.
			EX. 전월 대비, 전년 동월 대비, 전일 대비 등록세
			
[3] 수익현황
	1. 월 대여료
		1) 수익금액
			월 대여료의 수익금액 산정 방식에 대한 세부 수식 정의를 요청드립니다.
			수익율 %의 계산 수식과 기준 제공 요청 드립니다.
			
		2) 매각 잔가와 소매
			현재 저희측에서는 매각 차량을 구분할 수는 있지만, 해당 차량이 잔가 판매인지 소매 판매인지는 구분하고 있지 않습니다.
			소매 금액에 대한 관리도 진행되고 있지 않습니다.
			따라서 매각에 대한 세부 프로세스 정의가 진행되어야 해당 내용을 구현할 수 있을 것 같습니다.
			추가적인 로직에 대한 정의 부탁드립니다.
			
		3) 수익률
			수익률 계산 방식과 기준일 정의 부탁드립니다. 
			
[4] 보험 손해율
	1. 보험 손해율 산정 방식
		보험 손해율에 대한 세부적인 산정 방식이 누락되어 있는 것 같습니다. 답변 부탁드립니다.
		
[5] 미납 현황
	1. 정상, 주의, 심각 분류
		1) 기간 설정 
			요구사항 정의서 PPT 기준으로는, 정상(~30일)/주의(31일~60일)/심각(61일~)로 분류되어있는데,
			해당 기간은 어떤 정의에 의거한 것인지 궁금합니다.
			정산관리의 미수/연체와는 다른 개념일까요?
			
		2) 장기 채권 현황
			앞서 정산관리 변경안 질의에서 제안드렸듯이, 세부 기획안 논의 이후에 진행하는 것을 검토 부탁드립니다.
			

[6] 대여 손익 그래프
	
	해당 그래프는 렌터카 사업에서 활용되는 특수한 그래프라고 생각됩니다.
	일반적이지 않은 모델이기 때문에 상세한 그래프 로직을 제공부탁드립니다.
	
	1. 그래프 대상 범위
		해당 그래프의 대상 범위가 궁금합니다.
		렌트시스템 전사 포트폴리오에 대해서, 잔존가치 매각을 가정한 시뮬레이션 그래프일까요?
		만일 그러하다면 해당 부분에 대한 상세한 정의가 필요할 것 같습니다.
		
	2. 기간과 단위
		1) 시간 단위
			집계 단위 지정이 필요합니다. 연/분기/월(연 단위 그래프로 보이지만 월단위 막대그래프도 제공)
		2) 기간 
			a. 그래프 시작과 종료 기간 기준
				시작연~종료연에 대한 세부 규칙은 어떻게 되어야 할까요?
				차량 취득월이 기준인지, 계약 시작일 기준인지, 상세한 정의가 필요할 것 같습니다.
			b. 부분월 처리
				손익 월단위 처리에 있어 부분월 처리는 어떻게 처리해야할까요? (일할, 전액, 미집계)
				계산 방식에 다른 소수점 처리 규칙은 어떻게 될까요? (반올림, 내림 등)
			
	3. 데이터 정의
		1) 대여손익곡선
			a. 대여수익 포함 범위
				대여 수익은 어떤 값으로 산정하면 될까요? (월 대여료만? 혹은 기타 비용있는지)
			b. 부대비용의 데이터 정의
				부대비용 안에는 어떤 비용이 포함되어야 하나요?
				(정비, 비용, 세금 등... 상세한 내역)
			c. 감가상각 비용처리
				손익곡선에서 감가처리를 어떤 식으로 진행하는지 정의 부탁드립니다.
			d. VAT 처리 기준
				수익,비용,매각가에서 부가세 처리는 어떤 식으로 진행하시는지 정의 부탁드립니다.
			e. 엣지케이스 처리 방안
				중도해지, 사고전손, 장기 미가동 등에 대한 처리 방식을 정의해주시면 감사하겠습니다.
			
		2) 매각곡선
			a. 잔가 매각 곡선
				1) 매각 회계처리에서의 장부가 처리방식에 대한 정의가 필요합니다.
				2) 매각 시 발생하는 부대비용처리(탁송,상품화,수수료,세금) 방식에 대한 정의가 필요합니다. 
				3) 매각가는 만기인수금액 기반으로 처리하면 될까요?
			b. 소매가 매각 곡선
				매각 곡선은 잔가 매각을 기반으로 작성되는 것에 반해, 대여손익곡선은 소매가 매각을 기반으로 작성합니다.
				따라서 하기한 데이터에 대한 정의가 필요합니다.
					1) 데이터를 엔카에서 어떻게 가져와야 하는지 (API 계약 등)
					2) 데이터 산출은 월말가/월평균가 등 어떤 데이터를 선택해야하는지
					3) 차량 매핑은 어떻게 진행해야할지
				답변 부탁드립니다.
		
		3) 총 손익곡선
			a. 매각 곡선 반영
				현재 그래프에 필요한 매각 시나리오는 2가지 입니다.
					1) 잔가 매각 시나리오
					2) 소매가 매각 시나리오 - 대여손익곡선
				그렇다면 총 손익곡선은 어떤 매각 시나리오를 반영하여 작성해야할까요?
			b. 그래프 정의 혼선
				총 손익 그래프라고 정의되어 있는 그래프에 '매각 시나리오' 반영 여부가 정의되어 있지 않습니다.
				오히려 대여손익곡선 쪽에 '소매가 매각 시나리오'가 반영되어, 총 손익곡선과 유사한 정의가 되어있다고 판단됩니다.
				해당 그래프에 대한 명확한 정의가 필요할 것 같습니다.
				
		상기한 질의는, 요구사항 정의서에 작성된 단편적인 정보를 가지고 해석하여 드린 질문입니다.
		해당 곡선에 반영되어야하는 명확한 데이터 정의와 로직 정의를 적어주시면, 해당 내용을 추가 확인하고 재질의 드릴 수 있을 것 같습니다.
				
	4. 오차 허용치 및 QA 정의
		1) 그래프 오차 허용치
			해당 그래프가 예상 시나리오를 담고 있고, 실시간 반영 처리를 가정할 때, 오차 발생은 불가피한 영역입니다.
			따라서, 오차에 대한 허용치 합의가 필요합니다.
			연간, 월간에 따른 오차 허용치에 대한 정의가 필요합니다.
